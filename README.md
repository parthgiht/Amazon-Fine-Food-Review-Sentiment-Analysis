# Amazon-Fine-Food-Review-Sentiment-Analysis
Analyzes 568K+ Amazon reviews using VADER (NLTK lexicon-based) and RoBERTa (Hugging Face transformer). Achieves 85% accuracy with RoBERTa, comparing rule-based vs deep learning approaches through EDA, classification reports, and confusion matrix analysis. Built with Python, scikit-learn, NLTK, and Transformers.
